## Issues Found

### WARN-1: `/think on` is a toggle, not idempotent
- **Severity**: Low (cosmetic/UX)
- **Description**: `/think on` does not set think to ON; it toggles the current state. If think is already ON, `/think on` turns it OFF.
- **Impact**: Users expecting `/think on` to always enable thinking may be surprised.
- **Recommendation**: Consider making `/think on` idempotent (always enables) and `/think off` idempotent (always disables), or rename to `/think toggle`.

### WARN-2: Raw `<tool_call>` tags leak in chat/fin modes
- **Severity**: Medium (cosmetic)
- **Scenarios**: N011, N013, N020, N021
- **Description**: In chat and fin modes, the LLM sometimes generates `<tool_call>` XML tags attempting to use WebSearch tool. These raw tags are displayed to the user instead of being parsed/hidden.
- **Root Cause**: Chat/fin modes may not have tool execution enabled, but the LLM's system prompt still mentions tools, causing it to attempt tool calls that aren't intercepted.
- **Recommendation**: Either strip `<tool_call>` tags from output in non-tool modes, or adjust system prompts for chat/fin modes to not mention unavailable tools.
