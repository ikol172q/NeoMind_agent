═══════════════════════════════════════════════════════════════════════════════
                          CROSS-PERSONALITY MEMORY
═══════════════════════════════════════════════════════════════════════════════

WHAT PERSONALITIES CAN SHARE:
  ✓ User Preferences (any personality can read/write)
    - Example: Chat learns "timezone: UTC" → Finance uses for market hours
  
  ✓ User Facts (learned by one, accessible to all)
    - Example: Chat learns "work: SDE at Google" → Coding/Fin use for context
  
  ✓ Behavioral Patterns (accumulate across modes)
    - Example: Coding records "Python (3x)" → Finance sees language preference
  
  ✓ User Feedback (corrections propagate)
    - Example: Chat corrects "AAPL not APPL" → All modes learn from it

CONTEXT PRIORITIZATION:
  - Preferences: Equal priority (all shared)
  - Facts: Current mode's facts > shared facts > other modes' facts
  - Patterns: Current mode's patterns > shared patterns
  - Feedback: All modes see corrections equally

═══════════════════════════════════════════════════════════════════════════════
                            USAGE QUICK START
═══════════════════════════════════════════════════════════════════════════════

1. Import and initialize:
   from agent.memory import SharedMemory
   memory = SharedMemory()

2. Store preferences (any mode):
   memory.set_preference("timezone", "UTC", "chat")
   memory.set_preference("language", "en", "chat")

3. Remember facts:
   memory.remember_fact("work", "SDE at Google", "chat")
   memory.remember_fact("education", "BS CS from MIT", "chat")

4. Record patterns:
   memory.record_pattern("language", "Python", "coding")
   memory.record_pattern("frequent_stock", "AAPL", "fin")

5. Get context for LLM:
   context = memory.get_context_summary("coding")
   system_prompt = f"Base prompt...\n\n{context}"

6. Record feedback:
   memory.record_feedback("correction", "AAPL not APPL", "chat")

7. Retrieve memories:
   facts = memory.recall_facts("work")
   patterns = memory.get_patterns("language")
