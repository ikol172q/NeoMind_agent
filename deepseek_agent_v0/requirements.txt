## Installation instructions:

```bash
# Basic installation (minimum requirements)
pip install openai python-dotenv requests

# Full installation (recommended for all features)
pip install openai python-dotenv requests prompt_toolkit aiohttp lxml beautifulsoup4

# Or install from requirements.txt
pip install -r requirements.txt
```

## Package descriptions:

| Package | Purpose | Required |
|---------|---------|----------|
| `openai` | DeepSeek API client (OpenAI-compatible) | **Required** |
| `python-dotenv` | Environment variable management | **Required** |
| `requests` | HTTP library for API calls | **Required** |
| `prompt_toolkit` | Enhanced CLI with history, autocomplete | Optional (recommended) |
| `aiohttp` | Async HTTP client for efficient searching | Optional (recommended) |
| `lxml` | Fast HTML parser for search results | Optional (recommended) |
| `beautifulsoup4` | HTML parser (fallback) | Optional |

## Environment setup:

Create a `.env` file in your project root:
```env
DEEPSEEK_API_KEY=your_api_key_here
```

Get your API key from: https://platform.deepseek.com/api_keys
